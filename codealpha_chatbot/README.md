# Language Translation Tool

## Project Description
This is a Language Translation Tool developed using Python and Streamlit.

## Features
- Translate text between multiple languages
- Simple and user-friendly interface
- Supports English, Telugu, Hindi, Tamil, Kannada, French, Spanish, and German

## Technologies Used
- Python
- Streamlit
- deep-translator

## How to Run

pip install -r requirements.txt

streamlit run app.py
