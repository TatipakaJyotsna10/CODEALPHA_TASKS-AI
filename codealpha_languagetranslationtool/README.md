# 🌐 Language Translation Tool

## 📌 Project Description
The Language Translation Tool is a Python-based desktop application that allows users to translate text between multiple languages. It provides a simple and user-friendly graphical interface built with Tkinter and uses the `deep-translator` library to perform accurate translations through Google Translate.

This project was developed as part of the **CodeAlpha Artificial Intelligence Internship – Task 1**.

## ✨ Features
- User-friendly GUI using Tkinter
- Translate text between multiple languages
- Select source and target languages
- Fast and accurate translation
- Copy translated text with one click
- Simple and easy to use

## 🛠️ Technologies Used
- Python
- Tkinter
- deep-translator (Google Translator API)

## 📦 Required Library
```bash
pip install deep-translator
```

## 🚀 How to Run
1. Install Python (3.8 or above).
2. Install the required library:
   ```bash
   pip install deep-translator
   ```
3. Save the project as `language_translation.py`.
4. Run the program:
   ```bash
   python language_translation.py
   ```

## 🎯 Output
- Enter text in the input box.
- Select the source language.
- Select the target language.
- Click the **Translate** button.
- The translated text is displayed instantly.
- Use the **Copy** button to copy the translated text.
